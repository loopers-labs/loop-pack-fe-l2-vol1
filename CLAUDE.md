# 프로젝트 가이드

React 19 + Vite + TypeScript 기반 학습용 프론트엔드 과제 레포

## 명령어

- 개발 서버: `pnpm install && pnpm dev`
- 빌드: `pnpm build`
- 포맷: `pnpm format`
